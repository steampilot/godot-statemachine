# Design-Referenzen für CapricaGame

Sammlung von Games, die als Inspirationen für CapricaGame-Mechaniken dienen.

## Plattformer + Action-Flow

### Hades
- **Genre:** Roguelike Action
- **Relevanz:** Sidescroller Action-Flow, durchgehende Kämpfe ohne Breaks
- **Lern-Ziele:**
  - Smooth Combat mit vielen kleinen Kämpfen
  - Progression ohne lange Pausen
  - Rhythmische Feedback-Loop

### Katana ZERO
- **Genre:** Action Platformer
- **Relevanz:** Precision + Rhythmus, One-Hit-Kill Mechanik
- **Lern-Ziele:**
  - Minimale Controls (Slash, Dash, Parry)
  - Beat-basiertes Timing (wie Caprica mit Musik)
  - Schnelle, intensive Sequenzen

### Dead Cells
- **Genre:** Metroidvania Roguelike
- **Relevanz:** Exploration + Combat Flow, moderne Bewegungstools
- **Lern-Ziele:**
  - Bewegung als Kern-Mechanic neben Combat
  - Weapon-Variation (wie Caprica's Gitarren)
  - Smooth Movement-Feel

## Minimalistischer Combat

### Stick Fight: The Game
- **Genre:** Stick Figure Fighting
- **Relevanz:** Einfachste Controls (Punch, Kick), Physik-basiert
- **Lern-Ziele:**
  - Directe, reaktive Combat-Mechaniken
  - Keine komplexen Combos
  - Spaß durch Einfachheit

### Stick Warriors
- **Genre:** Side-Scrolling Fighter
- **Relevanz:** Minimal Design, schnelle Kämpfe
- **Lern-Ziele:**
  - Klare Visual Feedback ohne Clutter
  - Direkte Player-Input Response

## Rhythmus-Integration

### Crypt of the NecroDancer
- **Genre:** Roguelike + Rhythm
- **Relevanz:** Beat-basierte Bewegung, Musik als Core Mechanic
- **Lern-Ziele:**
  - Synchronisierung von Aktion zu Musik
  - Flow-State durch Rhythmus-Perfection
  - Musik verstärkt Gameplay statt Ablenkung

### Beat Saber (Konzept)
- **Genre:** Rhythm + Action
- **Relevanz:** Hit-on-Beat belohnt, optional aber incentivized
- **Lern-Ziele:**
  - Beat-Timing als Skill-Ceiling, nicht Requirement
  - Satisfying Feedback für präzise Timing

## Core-Learnings für CapricaGame

1. **Movement ist nicht alles** – Action-Flow ist das Ziel
2. **Einfache Controls** > Komplexe Combos (Stick Fight Style)
3. **Musik sollte Gameplay-Mechanic sein** (wie NecroDancer)
4. **Rhythmus als Belohnung, nicht Zwang** (wie Beat Saber)
5. **Smooth Feedback** wichtiger als komplexe Animation
